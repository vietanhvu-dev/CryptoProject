## 🔐 2. Privacy Policy

We value your privacy and commit to transparency regarding data handling:

### a. Zero Data Collection Policy
* **In-Memory Processing:** All inputs (Ciphertext, Keys, and Decrypted results) are processed temporarily in the application's memory (RAM) for the duration of the session.
* **No Persistence:** This application does not use any database or external storage to record your inputs or results.
* **Session Cleanup:** All data in the `session_state` is automatically purged when you close the browser tab or refresh the page (F5).

### b. Third-Party Infrastructure
This application is deployed on **Streamlit Cloud**. Please be aware that:
* Streamlit may collect basic system logs for technical maintenance and service stability.
* The developer has no control over, or access to, Streamlit’s internal infrastructure logs.

### c. Security Recommendations
* **Confidential Data:** Do **NOT** upload or analyze real-world sensitive data, including Personally Identifiable Information (PII), real passwords, or trade secrets, on this public cloud environment.
* **Open Source Audit:** You are encouraged to review the source code at [CryptoProject Repository](https://github.com/vietanhvu-dev/CryptoProject) to verify these security claims.
